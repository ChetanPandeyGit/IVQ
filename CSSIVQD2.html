CSS INTERVIEW QUESTIONS

Day 2

Q1. What is box sizing?


Ans. Box-sizing property defines how the width and height gets applied on an element.
The default value of box-sizing is content-box, where the actual width of the box will be equal to the border and padding and width of the box .
When box-sizing is set to be border-box, then the actual width will be equal to the width of the box itself, means if we will increase the padding or border of the box then it will not increase the width of the box.

Q2. What is animation delay?


Ans. It is a sub-property of CSS animation. It defines how long the animation has to wait before starting. Using the animation-delay property, you define the delay by specifying the number of seconds or milliseconds it should take before playing.

Q3. Which property you will use to merge cells vertically in grid?


Ans. grid-column-start and grid-column-end property can be used to merge cells vertically. It also has a shorthand i.e., grid-column: start/end.

Q4.Which property you will use to merge cells horizontally in grid?


Ans. grid-rows-start and grid-row-end property can be used to merge cells horizontally. It also has a shorthand i.e., grid-row: start/end.

Q5. what is the difference between display none and visibility hidden?


Ans. Display: none; property is used to hide elements without deleting them. It does not take up any space, which allows other elements to fill in.
Visibility: hidden; property also hides an element, but affects the layout i.e., it takes up the space.
