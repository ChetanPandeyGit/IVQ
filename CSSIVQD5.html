Q1. Show implementation of transition.

Q2. Show implementation of transformation.


Ans. CSS transforms are a collection of functions that allow to shape elements in particular ways:
translate: moves the element along up to 3 axis (x, y and z)
rotate: moves the element around a central point
scale: resizes the element
skew: distorts the element

Q4. What are CSS combinators ?


Ans. Combinators combine the selectors to provide them a useful relationship and the position of content in the document. Thisis most useful when you want to style certain elements all at once rather than styling elements individually. CSS combinator selectors are as follows.
Descendant Selector (space)
Child Selector (>)
Adjacent Sibling Selector (+)
General Sibling Selector (~)

Q5. What are pseudo selectors in CSS ?.


Ans. CSS pseudo-classes are used to add styles to selectors, but only when those selectors meet certain conditions. A pseudo class is expressed by adding a colon (:) after a selector in CSS, followed by a pseudo-class such as "hover", "focus", "active", "checked", "disabled", "visited", etc.
Syntax: 
        selector:pseudo-class {
                property : value;
            }
